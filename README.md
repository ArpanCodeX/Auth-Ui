# Auth UI Playground

A collection of beautiful authentication UI templates built with React and Tailwind CSS. Feel free to explore, customize, and use in your own projects.

## Getting started

1. **Clone the repository**
   ```sh
   git clone <YOUR_GIT_URL>
   cd <YOUR_PROJECT_NAME>
   ```
2. **Install dependencies**
   ```sh
   npm install
   ```
3. **Run the development server**
   ```sh
   npm run dev
   ```

You can use any IDE or editing workflow you prefer. This repo is a standard Vite React project.

## Technologies used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Build the project with `npm run build` and deploy the `dist` folder to any static hosting service (Netlify, Vercel, GitHub Pages, etc.).
